# Лендинг кофейни — «Brew Haven»

Демо-проект из портфолио [Lash](https://lash-devops.github.io/portfolio/). Атмосферный одностраничный лендинг для городской кофейни.

**Сложность:** 🟡 Средний
**Живое демо:** https://lash-devops.github.io/portfolio/projects/coffee-shop/

## Что это

Многосекционный контентный лендинг с тёплой палитрой, крупной типографикой и плавными анимациями. Показывает, как из контента (меню, фото, контакты) собрать цельную «продающую атмосферу».

## Секции

- **Hero** — полноэкранный экран с фото и призывом к действию
- **О нас** — текст + статистика
- **Меню** — две колонки (кофе и десерты) с ценами
- **Галерея** — сетка фото с **лайтбоксом** (клик → полноэкранный просмотр)
- **Как найти** — адрес, часы работы, телефон и встроенная карта OpenStreetMap

## Возможности

- 🍔 **Мобильное меню** (бургер) для навигации на телефонах
- 🖼️ **Лайтбокс** галереи с загрузкой увеличенной версии фото
- ⬆️ Кнопка **«наверх»**, появляющаяся при прокрутке
- 🗺️ Встроенная карта (iframe OpenStreetMap, без API-ключа)
- 🎬 Появление секций при прокрутке
- 📱 Полностью адаптивно (проверено от 320px)

## Технологии

- HTML5 (семантическая разметка)
- Tailwind CSS (Play CDN) с кастомной палитрой `coffee`
- Vanilla JavaScript — мобильное меню, лайтбокс, кнопка «наверх»
- Шрифты Google Fonts (Cormorant Garamond + Jost)

## Чему учит проект

Композиция длинной страницы, секционная вёрстка, работа с фоновыми изображениями и градиентами, несколько независимых интерактивных компонентов на чистом JS.

## Запуск

Открой `index.html` в браузере — сборка не нужна.

## Что можно заменить под себя

Название, тексты, позиции меню и цены, фотографии (ссылки Unsplash), адрес и координаты карты в `index.html`.
