# ОРБИТА — космический туроператор

Демо-проект из портфолио [Lash](https://lash-devops.github.io/portfolio/). Самый сложный из набора: многостраничный сайт со сложными анимациями и логикой бронирования.

**Сложность:** 🟣 Эксперт
**Живое демо:** https://lash-devops.github.io/portfolio/projects/space-travel/

## Что это

Сайт вымышленного космического туроператора: каталог направлений (орбита, Луна, Марс, Сатурн и др.), страницы деталей и многошаговый мастер бронирования с расчётом стоимости.

## Страницы (6)

| Файл | Страница |
|------|----------|
| `index.html` | Главная — герой, обратный отсчёт, статистика, избранные направления |
| `destinations.html` | Каталог с фильтром по типу и живым поиском |
| `destination.html` | Детали направления (рендер по `?id=`) |
| `booking.html` | Мастер бронирования из 4 шагов + экран подтверждения |
| `bookings.html` | Мои полёты — список подтверждённых броней с отменой |
| `about.html` | О компании — миссия, таймлайн, команда |

## Сложные анимации

- 🌌 **Звёздное небо на canvas** с мерцанием и параллаксом при прокрутке
- 🪐 **Планеты на чистом CSS** (радиальные градиенты, свечение, кольца Сатурна) — без картинок
- 🛰️ **Орбиты** с вращающимися спутниками (CSS keyframes)
- ✨ Появление блоков при скролле, 3D-наклон карточек, анимированные счётчики
- ⏱️ Живой **обратный отсчёт** до следующего запуска
- ✅ Анимированная галочка на экране подтверждения

## Сложная логика

- Каталог рендерится из данных (`assets/data.js`), фильтр по типу + поиск по ключевым словам
- Страница деталей собирается из `?id=` в URL
- **Мастер бронирования:** пошаговое состояние, валидация каждого шага, живой расчёт цены (`база × путешественники × класс каюты`), сохранение в `localStorage`, генерация номера брони
- **Мои полёты:** подтверждённые брони сохраняются в список (`orbita_bookings`), страница показывает их с итоговой сводкой и возможностью отмены
- Уважение к `prefers-reduced-motion` (анимации отключаются)

## Технологии

- HTML5, Tailwind CSS (Play CDN), Vanilla JavaScript, Canvas API
- Шрифты: Space Grotesk + Inter
- Без сборки и без внешних картинок — всё на CSS и canvas

## Структура

```
space-travel/
├── index.html, destinations.html, destination.html, booking.html, bookings.html, about.html
└── assets/
    ├── tw-config.js   — конфиг Tailwind (цвета, шрифты, анимации)
    ├── style.css      — общие стили, планеты, keyframes
    ├── data.js        — данные направлений и классов кают
    └── app.js         — общие скрипты (звёзды, reveal, счётчики, отсчёт, наклон)
```

## Запуск

Из-за общих файлов в `assets/` открывать нужно через локальный сервер (а не `file://`):

```bash
# из папки space-travel
python -m http.server 8080
# затем открыть http://localhost:8080
```
