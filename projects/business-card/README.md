# Сайт-визитка — «Алина Соколова»

Демо-проект из портфолио [Lash](https://lash-devops.github.io/portfolio/). Элегантная одностраничная визитка (vCard) специалиста.

**Сложность:** 🟢 Лёгкий
**Живое демо:** https://lash-devops.github.io/portfolio/projects/business-card/

## Что это

Минималистичная «цифровая визитка» с тёмной темой, эффектом стекла (glassmorphism) и мягкими анимациями. Подходит как личная страница-контакт для специалиста или фрилансера.

## Возможности

- 🎨 Тёмная элегантная тема, бирюзовый акцент, монограмма-аватар (без фото человека)
- 💾 **Скачивание контакта в vCard** (`.vcf`) — добавляется в телефонную книгу одним нажатием
- 📋 Копирование e-mail в буфер обмена по клику
- 🔔 Тост-уведомления о действиях
- 🎬 Появление при загрузке через `IntersectionObserver`
- 📱 Полностью адаптивно (проверено от 320px)

## Технологии

- HTML5 (семантическая разметка)
- Tailwind CSS (Play CDN)
- Vanilla JavaScript — генерация vCard через `Blob`, Clipboard API

## Чему учит проект

Базовая структура страницы, работа с Tailwind, простой JavaScript (события, Blob, скачивание файла) и анимации без библиотек.

## Запуск

Открой `index.html` в браузере — никакой сборки не требуется.

## Что можно заменить под себя

Имя, должность, описание, навыки и контакты в `index.html`; данные vCard — в блоке `BEGIN:VCARD` внутри `<script>`.
